# Code Of Conduct

## Our Standard

Project participation must remain respectful, technically honest, and focused on creating a safe collaborative environment. Constructive disagreement is welcome. Harassment, discrimination, threats, sexualized conduct, deliberate intimidation, doxxing, and sustained disruption are not acceptable.

## Scope

This policy applies in project repositories, discussions, issues, reviews, events, and public spaces where someone represents the project.

## Enforcement

Maintainers may edit or remove contributions and may warn, temporarily restrict, or permanently ban participants whose behavior violates this policy. Enforcement decisions should be proportionate, documented privately, and avoid conflicts of interest.

Report platform abuse through GitHub's private reporting tools. A dedicated project conduct-reporting channel must be confirmed before the project accepts broader external community operations; do not put sensitive personal information in public issues.

## Attribution

This policy draws on the Contributor Covenant 2.1 and GitHub Community Guidelines while using project-specific reporting language.

# Support

This release candidate has no support SLA.

Use GitHub Discussions for usage and design questions:

https://github.com/cadenzaio/cadenza-workspace/discussions

Use GitHub Issues for reproducible defects and bounded feature proposals:

https://github.com/cadenzaio/cadenza-workspace/issues

Security vulnerabilities must follow [SECURITY.md](./SECURITY.md) and must not be reported publicly. Legacy repositories, private managed-product features, Memory, CLI, and deferred UI/agent work are not supported by this repository unless its README explicitly says otherwise.
